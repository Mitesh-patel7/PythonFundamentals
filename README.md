# PythonFundamentals
This repository contains Python fundamentals assignment files.

Introduction:
  1. Assgnmt_Arithmetic_ops: Python script takes two user inputs for mathametical calculation (Addition, Multiplication, subtraction & Division)
  2. Assgnmt_Personalized_Greeting: Python script takes a user input to display a greeting message.
  3. Assgnmt2_Sum1-50 for loop: Python script calculates sum of all the integers from 1-50 using for loop.
  4. Assgnmt2 Even or Odd: Python script to get odd or even result as per the user's input.
  5. Assignment3 Factorial: Python script takes user input to calculate factorial of that number using recursive method
  6. Assignment3 Math Module: Python script takes user input to calculate square root, sin & log for a number.
